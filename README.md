# Base Vue Structure
Base project to start vue simple project

## Installation

#### Install dependencies
```bash
npm install
```

## Available commands

#### Development command
```bash
npm run serve
```

#### Production and compilation command
```bash
npm run build
```
#### Distribution Cleaning Command (delete dist folder)
```bash
npm run clean
```
